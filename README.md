# Rock Paper Scissors

The third project of **[The Odin Project](https://www.theodinproject.com/)** a Rock Paper Scissors web app where you play against a CPU and the first one to score 5 wins!
You can try it **[here](https://bishomilad.github.io/Rock-Paper-Scissors/)**

## Features

- Appealing UI
- A sketch-book theme
- Animations
- Rock Paper Scissors gameplay

## Learned Outcomes

- HTML basics
- CSS principles
- General styling
- UI/ UX
- DOM manipulation using JS
- CSS pattern design
